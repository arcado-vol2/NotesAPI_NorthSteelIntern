# Задние стажировки IT hub "северосталь"
Данный код реализует web-api заметок на C# фреймворк ASP.NET Core.
Используются библиотеки:
- MediatR
- autoMapper
- Swagger
- fluentvalidation
В качестве БД исользуется SQLite
